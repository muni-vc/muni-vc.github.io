⚡ Self-Initiated: Automation Suite: 8 Tools Built & Deployed

Entirely self-initiated. Built through AI-assisted development (vibe coding) using Python, Google Apps Script, and n8n, without a formal engineering background. Each tool was designed to solve a real, recurring problem in the team's workflow. Presented at a bi-weekly managers review and approved for full team adoption.

Taxonomy Bulk Drive File Renamer
Enforces client taxonomy naming standards and validates every attribute before renaming. Turns a tedious manual renaming task into a 2-minute automated process, eliminating AEM upload errors caused by non-compliant filenames.
Automated Drive File Renamer (Universal)
A general-purpose renamer built for every non-Amazon project in the agency. Handles full folder hierarchy, renames both files and folders, and applies AEM-standard naming logic. Converts what used to be a ~1 hour manual task into a 2-minute process with a 100% AEM upload success rate.
Amazon File Name Audit Tool
Acts as a safety firewall for high-volume Amazon projects. Scans up to 5 levels of folder hierarchy to detect AEM-illegal characters (spaces, +, parentheses), proposes compliant names automatically, and gives the user full control. Nothing is renamed without approval. Prevents upload delays caused by non-compliant filenames across hundreds of assets.
Project Setup Tool
A full project ecosystem automator that simultaneously builds the Drive folder structure and the brief spreadsheet in under 5 minutes. Handles both Channel Projects (automatically grouping into To Channel, Through Channel, and Internal categories) and Regular Projects. Replaces what previously took 3+ hours of manual setup.
Brief Drive Folder Creator
Automates the creation of standardized brief folders for every new project kickoff. Eliminates 15-20 minutes of manual folder creation per project, ensures every brief automatically contains the mandatory sub-structure (Assets, Exports, Source Files), and lets users pick destination folders visually without needing folder IDs.
Bulk Drive Folder Creator
Built for high-volume project setups. Creates 5 to 100+ main folders with custom hierarchy in 30 seconds, turning a 20-minute manual setup into an instant automated task, regardless of the depth or scale of the folder structure needed.
IDML Localization Tool
Automates the insertion of translated content directly into InDesign (IDML) files. Upload an IDML file and a translation Excel, select the target language, and the tool generates a fully localized IDML file along with a QA report, significantly reducing manual localization effort and turnaround time for design teams.
Visual Extraction Tool
Automates text extraction from PDF and PPTX files into a structured Excel format ready for translation workflows. Reads content in visual reading order, supports optional validation against existing files, and packages all outputs including a ZIP bundle, removing a major manual bottleneck in content preparation for localization.
